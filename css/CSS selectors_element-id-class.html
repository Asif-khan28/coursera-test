CSS selectors are used to determine which HTML element, or set of elements, to apply the CSS declarations to. The browser uses its Selector API to traverse the DOM s, and pick out the elements matching the selector.

Crafting a selector is a skill

Note:That's because a lot of JavaScript libraries out there ALSO use the browser selector API to attach behavior and data to HTML elements 


    //Element slector example:paragraph
    p{
      color:blue;
    }

    //class selector: can be use for several elements, so not unique
    .class_name{
      color:green;
    }
   <p class="class_name"> ......</p> 
   <div class="class_name"> ......</div> 

   //id selector: its unique
    #id_name{
      color:red;
    }
   <div id="id_name"> ......</div> 

   //group several css rules to one ; for Effciency
   //example
    div, .class_name{
      color:yellow;
    }
   <p class="class_name"> ......</p> 
   <div class="class_name"> ......</div>
