Subheading 1

CSS: CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. while p element is a selector, and has property and value in {}.

      color:blue;
      font-size: 20px;
      width: 200px;
      text-align:justify;
      word-spacing: 0cm;
    }
